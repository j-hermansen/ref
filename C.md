# C Programming Language


## Simple C program structure
Program name: `hello.c`
```C
#include <stdio.h>

int main() {
   printf("Hello, World!");
   return 0;
}
```

## Run C program
Compile
```bash
gcc hello.c
```
Run
```bash
./a.out
```
